Try out a modern browser. Here are some examples of good, modern browsers:

  • Google Chrome
  • Chromium
  • Microsoft Edge (the new version built on chromium)
  • Mozilla Firefox
  • Brave (make sure to turn of the "Shields Up" if you use this browser, otherwise this app might not work)
And please don't use Internet Explorer, as it doesn't support any new JavaScript anymore and probably never will be updated again...
In future this app will be compatible with more browsers.
